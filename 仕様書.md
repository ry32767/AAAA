# 迷路ゲーム仕様書

## 1. 概要
本ドキュメントは、Pythonで作成するGUI迷路ゲームの仕様を定義する。
このアプリケーションは、迷路の自動生成機能と、ユーザーが操作して迷路を解くプレイ機能を備える。

## 2. 主な機能
### 2.1. 迷路の自動生成
- アプリケーション起動時に、指定されたサイズの迷路を自動的に生成する。
- 生成される迷路は、必ずスタートからゴールまで到達可能な経路を一つ以上持つ。
- スタート地点は左上、ゴール地点は右下とする。

### 2.2. 迷路の表示
- GUIウィンドウ上に迷路を描画する。
- 壁、通路、プレイヤー、スタート、ゴールをそれぞれ異なる色で視覚的に表現する。

### 2.3. プレイヤー操作
- プレイヤーはキーボードの矢印キー（↑, ↓, ←, →）で操作する。
- プレイヤーは壁を通り抜けることはできない。
- プレイヤーが移動するたびに、画面表示を更新する。

### 2.4. ゴール判定
- プレイヤーがゴール地点に到達した場合、「ゴール！」などのメッセージを表示する。

## 3. 技術仕様
- **使用言語:** Python 3
- **GUIライブラリ:** `tkinter` (Python標準ライブラリ)
- **迷路生成アルゴリズム:** 穴掘り法（深さ優先探索を利用）
- **データ構造:**
    - 迷路: 2次元リストで壁と通路を管理する。
    - プレイヤー位置: `(x, y)` のタプルで座標を管理する。
